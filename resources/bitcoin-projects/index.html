Bitcoin Resources Bitcoin Projects

Useful bitcoin tools, utilities, and projects.

  • Bails

    Bails is a Bitcoin solution protecting against surveillance, censorship, and confiscation. It installs Bitcoin Core on the encrypted Persistent Storage of Tails, creates and recovers Bitcoin Core wallets from Codex32 (BIP93) seed backups, and creates backup Bails USB sticks and shareable blank Bails USB sticks.

  • DTails

    DTails is a tool that helps add Bitcoin software like Sparrow Wallet, Specter Wallet, Mempool.space, and others in Debian based live images like Tails.

  • Btcrecover

    An open source bitcoin wallet password and seed recovery tool designed for the case where you already know most of your password/seed.

  • Mnemonic Code Converter

    A tool for converting BIP39 mnemonic phrases to addresses and private keys.

  • Shamir Secret Sharing Scheme

    A tool for splitting secrets into parts or recreating secrets from existing parts.

  • Bitcoin Extended Public Key Converter

    Convert an extended public key from any version to a different version.

  • Cryptography Toolkit

    A web-based collection of cryptography tools for schemes/algorithms used in Bitcoin and LND.

  • HD Wallet Scanner

    Find all used addresses in your bitcoin hardware wallets bypassing gap limits.

  • Wallet Scrutiny

    The aim of this project is to improve the security of Bitcoin wallets by examining the application code for possible back-doors and other vulnerabilities.

  • Wallets Recovery

    Information about wallet defaults for external recovery.

  • 10x Security Bitcoin Guide

    How to store bitcoin without any single point of failure.
